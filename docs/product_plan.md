# Product Plan

## Direction

Ama-Pro should start as an internal tools platform, not a full SaaS product.
The first version should validate the complete workflow:

1. Open the tools platform.
2. Select a tool.
3. Upload input files or enter parameters.
4. Create a task.
5. Track task status.
6. Download results.
7. Review task history.

## First Milestone

Build a platform shell with mock tools and mock tasks.

Real modules can be plugged in later without redesigning the whole product.

## Later Milestones

- Add the Amazon ads diagnosis module.
- Add task persistence.
- Add file storage.
- Add user login if needed.
- Add more tools such as listing generation, image processing, keyword analysis, and competitor analysis.
